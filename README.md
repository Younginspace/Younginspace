# Hey, I'm Young

Builder. Vibe coder. Making things that feel alive.

I build interactive experiences — 3D worlds, AI conversations, cinematic web experiments — mostly with TypeScript, Three.js, and whatever AI model fits the vibe.

> 👉 **This repo is the source code for [younginspace.me](https://younginspace.me)** — my personal portfolio space station. Vibe-coded end-to-end with **Claude Code**, built on **Three.js + GSAP** for the 3D experience and **EdgeSpark** (Hono + Cloudflare Workers + React + Vite) for the stack. See [`server/`](./server) and [`web/`](./web).
>

## Running younginspace.me locally

```bash
cd server && npm install
cd ../web && npm install
edgespark deploy   # build + deploy (run from project root)
```

See [`CLAUDE.md`](./CLAUDE.md) for project layout and [`server/CLAUDE.md`](./server/CLAUDE.md) / [`web/CLAUDE.md`](./web/CLAUDE.md) for backend / frontend specifics.
