## 近期项目

### EdgeSpark — Agent-Native 后端 CLI
> 为 Vibe Coding 时代而生的一体化后端工具

一条命令完成部署、认证、数据库、存储——专为 AI 编程 Agent（如 Claude Code）设计。底层 Cloudflare Workers 边缘网络，`edgespark deploy` 即刻上线。可以理解为「Supabase + Vercel 合体」，让全栈开发不离开终端。

**技术栈**: Hono + Cloudflare Workers + Drizzle ORM + S3 Storage + OAuth

### Hail Mary Chat — 与外星人 Rocky 星际通话
> 《挽救计划》电影二创互动体验

打开页面，太空飞船通讯界面映入眼帘，Rocky 的 3D 全息投影漂浮在屏幕中央。点击通话，Rocky 会用他标志性的方式回应——「我是高兴！朋友！」AI 实时对话忠实还原书中 Rocky 的性格，支持中/英/日三语。

**技术栈**: React + Three.js + MiniMax AI (对话 + TTS) + 全息 Shader

### MOSS FATE — 与 MOSS 辩论人类命运
> 《流浪地球》哲学对话游戏

与数字意识体 MOSS 展开 4 轮哲学辩论：人类是否值得存在？穿越月球、地球轨道、极光等 7 个 3D 场景，最终投票决定人类命运——拯救或终结。还原 MOSS 标志性红眼呼吸灯动画和电影级打字效果。

**技术栈**: TypeScript + Three.js + i18n (中/英/日) + 7 个自定义 3D 场景

### Gosling Cinema — 高司令复古电影胶卷
> 致敬 Ryan Gosling 五部代表作的视觉体验

复古 CRT 电脑开机动画，镜头穿过屏幕进入 3D 胶卷旋转木马。Drive、银翼杀手 2049、爱乐之城、芭比、挽救计划——每部电影有独立的 Shader 后处理风格，鼠标移动触发液态扭曲效果。

**技术栈**: Three.js WebGL + EffectComposer + Lenis + GSAP + 自定义 GLSL Shader

### PixelMe — 像素风动态角色生成器
> LLM 驱动的像素艺术创作工具

上传参考图，AI 直接「画」出 32×32 像素角色（不是缩图！），自动检测骨骼关节，生成 idle/run/attack 三组动画。导出 GIF/PNG/Spritesheet，致敬 FF6、时空之轮等经典 JRPG。

**技术栈**: React + Claude Sonnet API + Canvas 骨骼动画 + gifenc 导出

## AIGC 愿景

1. **二创新范式** — AI 降低创作门槛，让更多人贡献创意，探索艺术形式的复用和覆盖
2. **交互升级** — 结合游戏 QTE、手势识别、陀螺仪等技术，创造更丰富的互动内容体验
3. **沉浸式未来** — 结合 AI 眼镜，探索 image-to-3D 的沉浸式消费，用深度信息创造新体验
4. **个性化供给** — 基于用户表达信号的生成式内容，补充传统供给短缺，增强个性化体验

**志向**: 更贴近 AIGC 应用场景和用户，深刻理解产品成败的本质原因。
